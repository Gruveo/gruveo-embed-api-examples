Gruveo Embed API: Basic Demo

This simple demo shows how you can add a Gruveo embed to your web application and interact with it using the provided JavaScript API. To establish a call, open this page in two or more tabs and join the same room in all. Open the JavaScript console in your browser to see the logs.

To see call recordings made with this demo, check the publicly accessible gruveo-api-demo-recordings S3 bucket.

A valid room name or a Gruveo @handle please.
